# [Dish Scanner Application 🍽️](https://subtle-hamster-632f37.netlify.app/)


Welcome to the **Dish Scanner** app, created during a **48-hour Hackathon**! This **end-to-end application** allows users to scan QR codes, calculate food calories in real-time, and access valuable health data from sources like the **World Health Organization (WHO)**. Developed using **React**, **Vite**, and **Firebase**, this project integrates real-time data interaction and dynamic data visualization.

## 🌟 Features

- **User Registration**: Users can sign up and log in securely to access all app features.
- **QR Code Scanning**: Scan QR codes to calculate food calories in real-time.
- **Dynamic Data Visualization**: Visualize calories data dynamically with **Recharts** using bar charts.
- **Admin Page**: Admins can perform CRUD operations to update data in real-time.
- **Know Why Page**: Provides insights sourced from **WHO** to educate users on health and nutrition.

## 🛠️ Technologies Used

- **Frontend**: 
  - React
  - Vite
  - Tailwind CSS
- **Backend**: Firebase Realtime Database
- **Libraries**:
  - Axios for API calls
  - Recharts for data visualization
  - React Router for page navigation
  - QRCode.react for QR code generation
  - react-qr-barcode-scanner for scanning QR codes

## 💡 Key Features & Insights

- **Real-time QR Scanning & Calorie Calculation**: Users can instantly calculate calories by scanning food QR codes.
- **End-to-End Application**: The app was fully developed in 48 hours, with real-time database interaction, secure authentication, and dynamic data visualization.
- **Data-Driven Health Insights**: The "Know Why" page integrates real-time data from **WHO** to educate users about nutrition and health statistics.
- **Admin Panel**: Enables admins to manage app data and perform CRUD operations.

## 🎯 Key Takeaways

- **Full-stack Development**: Integrated **React** for the frontend and **Firebase** for backend functionality, enabling seamless user interaction.
- **Real-Time Data Interaction**: Ensured that user interactions and data are dynamically updated in real-time.
- **Data Visualization**: Created engaging, dynamic charts using **Recharts** to display real-time food data.


## 🚀 **Hackathon Journey**

- **Timeframe**: 48 hours
- **Objective**: Build an end-to-end web application with real-time data interaction, secure authentication, and interactive features.
- **Outcome**: A fully functional QR code-based calorie scanner and an admin-controlled user management system with dynamic data visualization.
